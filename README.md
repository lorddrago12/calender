## ğŸ—ï¸ How It Works

### GUI Layout

The layout contains:

* An input field to display the selected date
* A button that opens the calendar
* A button that opens a date popup
* An exit button

### Event Loop

The program continuously:

1. Waits for user interaction
2. Detects which button was clicked
3. Opens a calendar popup if needed
4. Updates the input field with the chosen date
5. Displays confirmation messages

This is the standard structure of PySimpleGUI applications.
